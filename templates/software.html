VS Code
★ 4.9 下载 50M+ ↗ 持续增长
编辑器开源跨平台微软
🌱 生态介绍

拥有超过 5 万款扩展插件，覆盖几乎所有编程语言与框架。集成 GitHub Copilot、Codeium 等 AI 编程助手；通过 Remote-SSH / Containers / WSL 实现远程与容器内开发。Marketplace 生态成熟，主题、代码片段、调试器一应俱全。

⚡ 高阶用法

用 tasks.json 自定义任务流水线；launch.json 配置多目标调试；多根工作区隔离项目配置；Dev Containers 在容器中复现环境；Settings Sync 跨设备同步全套配置与插件。

🔗 联动方案

与 Docker（Dev Containers）、Git/GitHub（内置版本控制）、Postman（REST Client 扩展）深度联动；通过 Remote-SSH 连接服务器，配合 Tmux 实现持久会话。

Docker
★ 4.8 下载 30M+ → 稳定
容器DevOps部署跨平台
🌱 生态介绍

Docker Hub 托管数百万镜像，Compose 编排多容器应用。与 Kubernetes、Podman、BuildKit 构成完整容器生态。官方维护 MySQL/Redis/Postgres/Nginx 等常用镜像，社区贡献覆盖几乎所有主流服务。

⚡ 高阶用法

多阶段构建（multi-stage build）精简镜像体积；BuildKit 缓存挂载加速构建；compose profiles 管理多环境；healthcheck 与 restart policy 实现自愈；自定义桥接网络隔离服务。

🔗 联动方案

与 VS Code Dev Containers 无缝集成；配合 GitLab CI / GitHub Actions 构建 CI/CD 流水线；与 Postman 联调容器化 API；通过 compose + Nginx 反代部署多服务。

Postman
★ 4.6 下载 25M+ → 稳定
API 调试测试协作多协议
🌱 生态介绍

支持 REST / GraphQL / WebSocket / gRPC 多协议。内置脚本测试、Mock Server、环境变量与集合共享。团队 Workspace 支持协作，Public API Network 汇聚海量公开 API 模板。

⚡ 高阶用法

用 pre-request / test 脚本实现自动化断言与动态变量；Newman 命令行集成进 CI/CD；Collection Runner + 数据文件做参数化批量测试；Monitor 定时巡检 API 健康。

🔗 联动方案

与 VS Code REST Client 互补；导出 OpenAPI / Swagger 与后端框架对接；通过 Webhook 与 CI 系统联动做接口回归；配合 Docker 一键拉起依赖服务后联调。

#1 API 工具 访问官网 →
Git
★ 4.9 下载 100M+ → 行业标准
版本控制开源协作命令行
🌱 生态介绍

全球事实标准的分布式版本控制系统。围绕 Git 形成 GitHub / GitLab / Gitea 等托管平台，以及 GitKraken / SourceTree 等 GUI 客户端。GitHub Actions、GitLab CI 基于 Git 触发 CI/CD，几乎所有 IDE 原生集成。

⚡ 高阶用法

rebase 优雅整理提交历史；bisect 二分定位引入 Bug 的提交；worktree 同时检出多分支并行开发；reflog 找回误删提交；cherry-pick 跨分支移植改动；hooks 实现提交前自动检查。

🔗 联动方案
